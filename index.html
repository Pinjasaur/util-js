util.js – docs

  • Util.version returns the version of the library.
    • Util.version //
  • Util.author returns the author of the library.
    • Util.author //
  • Util.rot13(str) encodes/decodes str based on the rot13 cipher.
    • Util.rot13("Hello world") //
  • Util.squish(str) removes all whitespace from str.
    • Util.squish(" Hell o worl d ") //
  • Util.engrishify(str) trims and truncates all whitespace from str.
    • Util.engrishify(" Hello world ") //
  • Util.randHexColor(hash) returns a random hexadecimal color, with an optional hash appended to it.
    • Util.randHexColor() //
    • Util.randHexColor("#") //
  • Util.randInt(min, max) returns an integer between min and max.
    • Util.randInt(-100, 100) //
  • Util.fact(n) returns the factorial of n (-1 if n is < 0).
    • Util.fact(7) //
  • Util.escRegEx(regex) escapes the meta (special) characters from the regular expression regex.
    • Util.escRegEx("[^$.*/\\]") //
  • Util.repAll(str, find, rep) finds and replaces all instances of find in str with rep.
    • Util.repAll("Will Smith will smith while smithing his will", "will", "code") //
  • Util.maskify(str, mask, offset) replaces all but the last 4 characters of a string with "#". Optional parameters of mask for changing the masking character and offset for changing the offset value.
    • Util.maskify("util.js - a js utility library") //
    • Util.maskify("util.js - a js utility library", "$", 7) //
