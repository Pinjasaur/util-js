util.js
a js utility library by @Pinjasaur.

util.js – docs

  • Util.version returns the version of the library
    • example: Util.version //
  • Util.author returns the author of the library
    • example: Util.author //
  • Util.rot13 encodes/decodes a string based on the rot13 cipher
    • example: Util.rot13("Hello world") //
  • Util.squish removes all whitespace from a string
    • example: Util.squish(" Hell o worl d ") //
  • Util.engrishify trims and truncates all whitespace from a string
    • example: Util.engrishify(" Hello world ") //
